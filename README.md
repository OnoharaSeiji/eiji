# Objetivo Eiji

Estrutura pronta para publicar no GitHub Pages o plano de estudos do Eiji para o Desafio Objetivo.

## O que já vem pronto
- cronograma completo de **214 dias**
- navegação por mês
- páginas diárias geradas a partir do cronograma
- apostila completa do **Dia 1**
- dados do cronograma em JSON
- pasta separada para respostas e exemplos
- scripts base para regenerar páginas no futuro

## Como publicar
1. Crie um repositório novo no GitHub
2. Envie todo o conteúdo deste zip
3. Vá em **Settings > Pages**
4. Escolha:
   - **Deploy from a branch**
   - **Branch: main**
   - **Folder: /docs**
5. Salve

A home fica em `docs/index.html`.
